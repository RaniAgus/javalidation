# javalidation

Example library for validating Java code.

## Installation

1. Add this to `pom.xml`:
```xml
    <dependency>
      <groupId>io.github.raniagus</groupId>
      <artifactId>javalidation</artifactId>
      <version>0.1.0</version>
    </dependency>
```

2. Run via command line:

```bash
mvn install
```
